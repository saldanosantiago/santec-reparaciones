# Santec — Reparaciones

Sitio web profesional para Santec, servicio técnico especializado en reparación de celulares en Catamarca, Argentina.

## 🔗 Demo en vivo

[santec-reparaciones.netlify.app](https://santec-reparaciones.netlify.app/)

## 📸 Preview

![Preview de Santec](img/preview.png)

## 🛠 Tecnologías

- HTML5 semántico
- CSS3 con variables personalizadas y Flexbox/Grid
- JavaScript vanilla
- Font Awesome para íconos
- Git y GitHub para control de versiones
- Netlify para deploy automático

## 📌 Características

- Diseño responsive mobile-first
- Sistema de variables CSS para consistencia visual
- Menú hamburguesa funcional en móvil
- Smooth scroll entre secciones
- Integración directa con WhatsApp
- Metadatos Open Graph para compartir en redes
